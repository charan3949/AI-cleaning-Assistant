# AI Cleaning Operations Assistant

An AI-powered assistant that helps cleaning and facility management teams analyze and respond to service issues efficiently.



# Features
- Classifies cleaning/service issues
- Assigns priority levels (Low, Medium, High)
- Suggests actions for resolution
- Generates professional responses for managers



# Tech Stack
- Python
- OpenAI API



# Example

INPUT:
Cleaning missed on 2nd floor and client complained twice

OUTPUT:
- Issue Category: Cleaning Issue
- Priority: High
- Suggested Action: Immediate cleaning and client follow-up
- Professional Response generated



# Business Value
- Reduces manual communication
- Improves response time
- Helps prioritize critical issues
- Useful for cleaning & facility management companies



# How to Run

1. Install Python  
2. Install dependencies:
   pip install openai  
3. Add your API key  
4. Run:
   python app.py
